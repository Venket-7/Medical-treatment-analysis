# Medical Treatment Cost Analysis & Prediction

## Project Overview
This project analyzes healthcare data to understand treatment costs, disease distribution, and patient demographics using machine learning techniques.

## Objectives
- Perform Exploratory Data Analysis (EDA)
- Analyze healthcare treatment patterns
- Predict treatment costs
- Classify diseases using machine learning

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Workflow
1. Data Understanding
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Engineering
5. Model Building
6. Model Evaluation

## Machine Learning Models
### Random Forest Regressor
Used for:
- Treatment Cost Prediction

### Random Forest Classifier
Used for:
- Disease Classification

## Evaluation Metrics
### Regression
- R² Score
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- Mean Absolute Error (MAE)

### Classification
- Accuracy Score
- Classification Report


## Key Findings
- Treatment costs vary across patient and disease characteristics.
- Feature engineering improved model readiness.
- Random Forest identified important predictive features.

## Limitations
- Dataset may not fully represent real-world healthcare scenarios.
- Additional medical features may improve performance.
